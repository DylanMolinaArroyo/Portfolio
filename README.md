# Portfolio

Personal portfolio website built with **Angular 19**, featuring SSR, PWA support, and bilingual content (English / Spanish).

## Tech Stack

- Angular 19 + SSR
- ngx-translate (i18n)
- Angular Service Worker (PWA)

## Getting Started

```bash
npm install
npm start        # dev server at http://localhost:4200
npm run build    # production build
```

## Features

- English / Spanish language switcher
- Sections: Home, About, Education, Certifications, Experience, Projects, Skills
- PWA-ready with web manifest and service worker
